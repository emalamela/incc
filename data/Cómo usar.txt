Para generar imágenes: La variable problem produce imágenes del problema 1 si es 1 y del problema 2 si es 2. Si se abre el script en processing y se corre, la primera imagen sale en blanco (no válida) y después cada vez que se hace click la imagen que estás viendo se guarda y se genera una nueva. Si apretás la flecha derecha generás una nueva imagen sin guardar la que estás viendo (esto está bueno para controlar que el set de imágenes final no tenga imágenes que no nos gusten).

La idea eventualmente sería con esto construir también los casos difíciles, manteniendo este código pero agregando más variables (habría que pensar cuáles). También pongo en duda si esto no será quizás demasiado fácil. Por ejemplo en el problema 1 las formas son solamente elipses o rectángulos, cuando en bongard #35 son todo tipo de formas. Podrían crearse esas formas como imágenes y después usarse con el mismo código. Pero esto al menos sirve como set de imágenes inicial para probar cómo anda.
